# Frank Fried Kitchen — WhatsApp Order Bot

A live WhatsApp ordering system built for **Frank Fried Kitchen**, a food vendor in  Accra, Ghana.
🌐 **Live Demo:** [waakye-bot.onrender.com](https://waakye-bot.onrender.com)

## What It Does
- Customers message the business on WhatsApp and receive an interactive menu with clickable buttons
- Browse food categories (Rice Dishes, Pasta, Specials, Drinks)
- Add items to a cart, choose pickup or delivery, select payment method (MoMo or Cash)
- Owner receives instant notification for every order
- Built-in credit sales tracking with automatic payment reminders

## Tech Stack
- **Backend:** Python, FastAPI
- **Messaging:** WhatsApp Cloud API (Meta)
- **Database:** SQLite
- **Deployment:** Render (free tier)

## Impact
- Reduced manual order-taking time for the owner
- Zero missed orders during peak cooking hours
- First week free trial converted to paid monthly client

## Status
✅ Live and deployed on Render. Currently serving customers in Accra.
